## 🖼️ Simple Image Classification using CNN

This is a beginner-friendly deep learning project where I implemented a Convolutional Neural Network (CNN) for image classification.  
The main purpose of this project is learning and understanding how CNNs work.

---

## 📌 About

This project is part of my learning journey in Deep Learning.  
I built a simple CNN model to classify images using the MNIST dataset.

---

## 📊 Dataset

- MNIST dataset  
- Contains handwritten digits (0–9)  
- Used for training and testing the CNN model  

---

## 🚀 What I Learned

- Basics of CNN (Convolutional Neural Networks)  
- How convolution and pooling layers work  
- Model training and evaluation  
- Working with image datasets like MNIST
